My first week at Great Wolf Resorts, I was handed the project of unifying security dashboards in Rapid7 — pulling CrowdStrike, FireEye, and Microsoft Defender data into a single view. I had no background in any of those platforms. It was a three-month timeline, and I spent the early weeks leaning on a more experienced engineer for guidance while I learned the APIs and data models. The end result gave the team visibility they'd never had: correlated threat data across all their security tools in one place instead of four separate consoles. That project became the template for much of my later work: take disparate data sources, figure out how they relate, and build the unified view that lets people actually make decisions. A concrete problem gave me a faster path into the domain than abstract documentation study would have.
