I use AI tools aggressively in my personal projects — building locally-hosted SaaS apps, home lab experiments, daily coding with Claude Code and Cursor. That means I've hit the failure modes firsthand: an AI agent wiped an entire database, another deleted a project folder. When my team at UCOP started experimenting with LLM-based tooling, I'd already lived through the disasters that happen without guardrails. So I built the governance framework before leadership asked — review standards, audit trails, operational boundaries. The pitch wasn't "slow down," it was "here's what goes wrong without this, and I know because it happened to me." That credibility from personal experience made it an easy sell.
